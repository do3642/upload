역량을 갖추는 것이 저의 목표입니다. 트렌드를 읽고, 기술을 깊게 이해하며, 다양한 관점에서 문제를 해결하는 개발자로 성장하고 싶습니다.
What Can i Do?
클라이언트와 서버를 모두 다룰 수 있는 풀스택 환경에서, 요구사항 분석부터 API 설계, 프론트 구현까지 전체 흐름을 고려한 개발을 지향합니다. 프론트엔드에서는 반응형 웹과 시멘틱 마크업에 신경 쓰며, 백엔드에서는 Flask와 Spring Boot 기반 REST API를 구축하고, 데이터베이스 설계를 포함한 웹 사이트 팀프로젝트 경험이 있습니다. 새로운 기술을 배우는 것과 팀 피드백을 통해 성장하는 것을 즐깁니다. 유연하고 확장 가능한 구조를 고민하며, 실무에서도 바로 적용 가능한 코드를 작성하기 위해 노력하고 있습니다. 반응형 웹 / 시멘틱 마크업 JavaScript & jQuery & React RESTful API 설계 (Flask / Spring Boot) 데이터베이스 설계 및 연동 Git & 협업 중심 개발
